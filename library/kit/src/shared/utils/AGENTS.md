# AGENTS.md

## Что это

Зарезервированный каталог для будущих низкоуровневых shared utilities.

## Назначение

Сейчас каталог не содержит утилит и не участвует в public API.

Он оставлен только как возможная точка для будущих общих helpers, если появится реальное повторное использование несколькими срезами.

## Как реализован

- Public `index.ts` отсутствует.
- Source-файлов нет.
- Каталог не должен импортироваться как слой, пока здесь нет согласованных утилит.

## Public API и локальные файлы

- Public API отсутствует.
- Нельзя экспортировать этот каталог из `@tiyn/kit`.

## Фактическое поведение

- Runtime behavior отсутствует.
- Сборка не должна зависеть от этого каталога.

## Зависимости

- Зависимости отсутствуют.
- Будущие utilities не должны импортировать `core`, `icons` или `features`.

## Правила изменений

- Не добавлять сюда helper, который нужен только одному компоненту.
- Не создавать `index.ts` без отдельного решения.
- Если утилит не появится, каталог можно удалить отдельной cleanup-правкой.

## Риски и точки внимания

- Основной риск — превратить `shared/utils` в неструктурированную свалку.
- Каждый новый файл должен иметь понятного потребителя и описанную ответственность.

## Технический долг

- Каталог пустой и зарезервированный. Его стоит удалить, если в ближайших этапах не появятся реальные shared utilities.

## Проверка

- Если здесь появится TypeScript-код, запускать `../../node_modules/.bin/tsc -p tsconfig.json --noEmit --incremental false`.
