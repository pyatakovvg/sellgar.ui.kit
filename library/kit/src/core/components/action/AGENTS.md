# AGENTS.md

## AGENTS узла

Путь: `src/core/components/action`

## Что это

Family-каталог action-компонентов.

## Назначение

Здесь размещаются компоненты, которые инициируют пользовательское действие: кнопки, группы кнопок и stepper-управление.

## Правила изменений

- Экспортировать компоненты через `index.ts` family-узла.
- Не добавлять сюда поля ввода, навигацию или status/content-компоненты.
- Внутренние parts держать внутри каталога конкретного компонента.

## Проверка

- Для source-изменений запускать `../../node_modules/.bin/tsc -p tsconfig.json --noEmit --incremental false`.
