# AGENTS.md

## AGENTS узла

Путь: `src/core/systems`

## Что это

Структурный каталог общих runtime-систем core UI.

## Назначение

Здесь размещаются переиспользуемые механизмы, которые не являются самостоятельными публично воспринимаемыми компонентами.

## Правила изменений

- Публичные компоненты не размещать в `systems`.
- Система должна уменьшать связность нескольких компонентов, а не быть переносом локального helper одного компонента.
- Экспортировать системы через `index.ts`.

## Проверка

- Для source-изменений запускать `../../node_modules/.bin/tsc -p tsconfig.json --noEmit --incremental false`.
